# customer-churn-analysis

Problem Statement - 
Customer churn leads to significant revenue loss in subscription-based businesses.
This project aims to identify customers at risk of churn and recommend strategies to improve retention.

Key Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Churn Prediction using ML models
- Business Insights & Recommendations

Model Performance
- Logistic Regression ROC-AUC: ~0.83
- Random Forest ROC-AUC: ~0.81

Business Impact
- ~7% customers identified as high-risk
- Estimated revenue at risk: ₹2.3L/month

Tech Stack - 
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
